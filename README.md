# Flask Api Documentation 

- with Swagger & ReDoc

### Install Packages
pip install -r requirements.txt

### Run Command
```flask run```

### Dot-Env Format (.env)
```FLASK_SWAGGER_URL='/api/swagger'
FLASK_API_URL='/static/api_doc.json'
```

### Documentation endpoints
* Swagger : ```/api/swagger```
* ReDoc : ```/api/redoc```
